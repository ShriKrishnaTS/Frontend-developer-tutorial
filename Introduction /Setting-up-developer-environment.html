Developer Environment

For any developer, it is most important to be in a right developer environment. Developer environment refers to the procurement of all the necessary tools, editors, browsers etc before diving deep into development mode (Trust me, development mode is a super natural power for any developer, powerful than the beast mode :) )

A front-end developer would require two tools to develop a webpage.

  • Browser
  • Editor

Browser - This is to view the webpage that you code using an editor. There are n number of browsers available. As per the recent study, chrome has the highest number of users and I personally recommend you to use this browser if you are not used to it. We will be using the chrome web browser in this tutorial

Editor - This is a software to write your code. You can use any editor available today such as, notepad, notepad++, gedit(linux users), sublime text, rubymine, atome etc. I personally recommend users using sublime since it has all the required advanced features. Also, it is available for free download. You can find a liscece key on github and use this editor for lifetime without any worries
