HTML, CSS, JS put together

HTML

  • HTML stands for HyperText Markup Language
  • HTML generally defines the structure of a webpage like-
    • Add an image to the page
    • Add a form to the page
    • Add a paragraph to the page
  • HTML is called skeleton of a webpage

CSS

  • CSS stands for Cascading Style Sheets
  • CSS generally defines the style for HTML such as
    • Add a black border of 2px for an image
    • Add background color to the form added
    • Make all paragraphs yellow
  • CSS is called the skin on the skeleton

JS

  • JS stands for JavaScript
  • JS generally adds logic and interactivity to a webpage such as
    • Change border color of an image when the user ckicks on it
    • Validate the phone number field on form
    • Change paragraph color on hover
  • JS is called the action of a webpage
